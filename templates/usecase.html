Capability Matrix
Realtime Operations
  • Live multi-camera dashboard with overlay detection boxes
  • Camera health widget, model status, and detection summary widgets
  • Quick fullscreen and stream quality controls
Camera + ROI Control
  • RTSP onboarding with credentials and per-camera model settings
  • Detection classes by zone, always-active or scheduled ROI modes
  • Email and Gemini-enhanced notification toggles by ROI
Playback + Evidence
  • Timeline scrubbing with recording and detection event tracks
  • Object filters and event-only review mode
  • Snapshot and video download actions for incident records
Face Intelligence
  • Face profile library with sample management and naming
  • Face detector model/provider tuning and confidence controls
  • Recognition grouping and optional auto-assimilation workflows
System Reliability
  • CPU, memory, disk, and optional GPU telemetry
  • Resource charts for trend-based capacity planning
  • Centralized log-level and retention tuning from settings
Security + Admin
  • User profile controls and preference management
  • Admin user management and AI model administration
  • Role-aware navigation and configuration controls
Example Deployment Playbooks
Warehouse Security

Reduce blind spots and unauthorized access in loading and storage areas.

  • Set intrusion ROIs on restricted aisles
  • Enable after-hours email alerts
  • Use playback timeline for shift-level investigations
Corporate Campus

Track lobby, corridor, and parking flow with proactive incident triage.

  • Use person + vehicle class-specific ROI rules
  • Leverage face grouping for repeated unidentified visitors
  • Monitor system load to plan camera expansion
Retail Operations

Improve safety and response for entrances, counters, and storage rooms.

  • Deploy loitering and restricted-zone detection
  • Use Gemini summaries for faster alert interpretation
  • Keep event snapshots for compliance documentation
